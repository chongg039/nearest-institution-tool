# 企业最近机构匹配

纯静态 GitHub Pages 工具，用于把企业注册地址匹配到最近机构网点。

同一套页面也可以通过 Electron 打包成 Windows、macOS、Linux 桌面端应用。

## 数据安全

- 不内置任何本地企业或机构数据。
- 不读取环境变量。
- 不使用服务器端缓存。
- 不使用 `localStorage`、`IndexedDB` 或 Cookie。
- 高德 Key 由用户在页面显式输入，只保存在当前页面内存中。
- 上传的 CSV/Excel 和生成结果只存在于当前浏览器会话内。

## 使用

1. 输入高德 Web 服务 Key。
2. 批量匹配时上传企业 CSV/Excel 和机构 CSV/Excel。
3. 企业文件至少包含 `注册地址`。
4. 机构文件至少包含 `机构地址` 和 `网点`。
5. 输出会新增或覆盖：
   - `最近机构地址`
   - `最近机构网点`
   - `企业到网点直线距离(米)`

距离是基于高德坐标计算的直线距离，不是驾车或步行距离。

## 已解析数据展示

上传已解析完成的 CSV/Excel 后，页面会展示：

- 企业分布 Top
- 网点粒度完整排行
- 数据预览

## 周度指标驾驶舱

页面顶部可切换到“周度指标驾驶舱”。该模块支持上传 Excel/CSV，也支持生成 8 周、10 个机构的模拟数据。

指标文件要求：

- `机构` 必填，也兼容 `机构名称`、`网点`、`支行`。
- `周次` 可选，没有时按 `本周` 处理。
- 可识别 `综合业务覆盖率`、`电票业务穿透率`、`收单业务覆盖率`、`代发工资业务覆盖率`、`国业业务穿透率`、`高渗透客户占比`、`对公结算业务`、`对公贷款业务`、`综合业务备款贡献度`、`综合付息率`、`重点客群数`。
- Numbers 文件请先导出为 `.xlsx` 或 `.csv` 后上传。

驾驶舱默认展示在最近机构匹配前面，包含重点客群综合业务渗透总体情况、机构前五/后五排行、可展开的中间机构排行、周度趋势、单一机构分析和业务短板热力图。各板块提供细分或指标选择控件；周度趋势支持折柱同图、雷达图、玫瑰图、热力图，并统一按所选年份的年初周次做新增对比；热力图会用圆点标出每家机构自己的低分业务。

## 桌面端开发

安装依赖：

```bash
npm install
```

启动桌面端：

```bash
npm run desktop
```

本机目录打包：

```bash
npm run pack
```

生成安装包：

```bash
npm run dist
```

打包产物默认输出到 `release/`，不会提交到仓库。
